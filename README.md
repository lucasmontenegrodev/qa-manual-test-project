# Projeto de Testes Manuais – QA

Este repositório contém casos de teste manuais criados para validar uma aplicação de treinos desenvolvida em Python.

## Objetivo
Demonstrar conhecimentos em QA manual, incluindo:
- Escrita de casos de teste
- Documentação de bugs
- Uso do Jira
- Versionamento com GitHub

## Ferramentas utilizadas
- Jira
- GitHub
- Python
- Markdown

## Estrutura do projeto
casos-de-teste/  -> Casos de teste e bugs  
evidencias/     -> Evidências (prints do Jira)

## Aplicação testada
Programa em Python para cálculo de periodização de treino (1RM).

Repositório do código-fonte:
https://github.com/lucasmontenegrodev/logica-programacao-python

Arquivo testado:
Projetos/calculadora_periodizacao_1rm.py
