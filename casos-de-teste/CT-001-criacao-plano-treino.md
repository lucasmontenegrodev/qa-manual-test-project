# CT-001 – Criação de plano de treino com dados válidos

## Objetivo
Validar a criação de um plano de treino com dados válidos informados pelo usuário.

## Sistema testado
Aplicação de treinos desenvolvida em Python (execução via terminal).

## Pré-condições
- Aplicação em execução
- Usuário informa dados válidos

## Dados de teste
- Dia de treino: Segunda-feira
- Quantidade de exercícios: 1
- Exercício: Supino
- 1RM: 100 kg

## Passos
1. Executar a aplicação no terminal
2. Informar os dias de treino
3. Informar a quantidade de exercícios
4. Informar o nome do exercício
5. Informar o valor do 1RM

## Resultado esperado
O sistema deve gerar corretamente o plano de treino com cargas e repetições.
