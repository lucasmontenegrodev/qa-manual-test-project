# BUG-001 – Sistema aceita valor negativo para 1RM

## Descrição
O sistema permite inserir valores negativos para o 1RM.

## Passos para reproduzir
1. Executar o programa
2. Informar valor negativo no campo 1RM

## Resultado esperado
O sistema deve bloquear valores negativos.

## Resultado obtido
O sistema aceita o valor e gera o plano de treino.
